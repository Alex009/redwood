class WeakBindingTrace

This class is meant to have the shape of a BindingTrace object that could exist and flow through the Psi2Ir -> Ir phase, but doesn't currently exist. Ideally, this gets replaced in the future by a trace that handles this use case in upstream. For now, we are okay with this because the combination of IrAttributeContainer and WeakHashMap makes this relatively safe.
