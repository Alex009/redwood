class DurableKeyVisitor(keys: MutableSet<String>)

This data structure is used to build unique but durable "key paths" for tree structures using a DSL.

This is primarily used by the LiveLiteralTransformer to create unique and durable keys for all of the constant literals in an IR source tree.
