fun buildPath(prefix: String, pathSeparator: String = "/", siblingSeparator: String = ":"): Pair<String, Boolean>
Content copied to clipboard

Build a path at the current position in the tree.

Return

A pair with first being the built key, and second being whether or not the key was absent in the dictionary of already built keys. If second is false, this key is a duplicate.

Parameters

prefix

A string to prefix the path with

pathSeparator

The string used to separate parts of the path

siblingSeparator

When duplicate siblings are found an incrementing index is used to make the path unique. This string will be used to separate the path part from the incrementing index.
